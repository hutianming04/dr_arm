# 一阶低通滤波器参数：α、τ 与截止频率

## 1. 离散公式

$$y[k] = \alpha \cdot x[k] + (1-\alpha) \cdot y[k-1]$$

- $\alpha$ — 滤波系数（0~1），**越大响应越快、滤波越弱**
- $x[k]$ — 当前输入
- $y[k-1]$ — 上一次输出

---

## 2. 连续域对应关系

连续域一阶低通的传递函数：

$$H(s) = \frac{1}{\tau s + 1} = \frac{\omega_c}{s + \omega_c}$$

其中 $\omega_c = 2\pi f_c$ 是截止角频率。

用**后向欧拉法**离散化（$s \to \frac{1-z^{-1}}{T_s}$）：

$$\boxed{\alpha = \frac{T_s}{T_s + \tau}} \quad\quad \boxed{\tau = T_s \cdot \frac{1-\alpha}{\alpha}}$$

---

## 3. α 与截止频率 $f_c$ 的关系

| 关系 | 公式 |
|---|---|
| α → fc | $$f_c = \frac{\alpha}{2\pi T_s (1-\alpha)}$$ |
| fc → α | $$\alpha = \frac{2\pi f_c T_s}{1 + 2\pi f_c T_s}$$ |

**实例**（$T_s = 1\text{ms}$）：

| α | fc | 效果 |
|---|---|---|
| 0.001 | 0.16 Hz | 极强滤波，响应极慢 |
| 0.005 | 0.8 Hz | 强滤波 |
| 0.01 | 1.6 Hz | 较强滤波 |
| 0.05 | 8.4 Hz | 中等滤波 |
| 0.1 | 17.7 Hz | 一般滤波 |
| 0.3 | 68 Hz | 轻度滤波 |
| 0.5 | 159 Hz | 很轻滤波 |
| 0.8 | 637 Hz | 几乎不过滤 |
| 0.9 | 1432 Hz | 几乎不过滤 |
| 1.0 | ∞ | 直通（不滤波） |

---

## 4. τ 的物理意义

**τ 是时间常数**，定义为滤波器对阶跃输入响应达到 **63.2%**（$1-1/e$）所需的时间。

| 时间点 | 响应百分比 | 说明 |
|---|---|---|
| $t = \tau$ | 63.2% | 一个时间常数 |
| $t = 3\tau$ | 95.0% | 接近稳态 |
| $t = 5\tau$ | 99.3% | 基本达到稳态 |

- **τ 小** → 响应快、噪声滤除少、相位滞后小 → 适合跟踪动态信号
- **τ 大** → 响应慢、平滑性强、相位滞后大 → 适合滤除高频噪声

---

## 5. 如何取值

### 5.1 从截止频率出发

先确定需要的截止频率 $f_c$，再计算：

$$
\tau = \frac{1}{2\pi f_c}
$$

$$
\alpha = \frac{2\pi f_c T_s}{1 + 2\pi f_c T_s}
$$

### 5.2 各场景建议值

| 场景 | τ 取值建议 |
|---|---|
| **PID 微分滤波** | $\tau \approx (0.1\sim0.5) \times \frac{K_d}{K_p}$，或 $\tau = \frac{1}{2\pi f_{noise}}$ |
| **前馈目标滤波** | $\tau$ 取系统主导时间常数的 $1/10 \sim 1/5$ |
| **传感器信号滤波** | 根据 $f_c$ 反算。如保持 10 倍采样率余量：$f_c \leq \frac{f_s}{10}$ |
| **编码器速度滤波** | $\tau = 1\sim5\text{ ms}$（一般）或 $\tau = 10\sim50\text{ ms}$（强滤波） |
| **电流环反馈** | $f_c = 200\sim500\text{Hz}$ → $\tau \approx 0.3\sim0.8\text{ ms}$ |

### 5.3 快速经验法

> $\tau = (5\sim20) \times T_s$
>
> - 需要强滤波取大端（如 20×Ts）
> - 需要快速响应取小端（如 5×Ts）
> - 一般从 $\tau = 10 \times T_s$ 起步试调

---

## 6. 代码对应关系

你的 `filter.c` 中初始化的换算：

```c
// 已知 tau 求 alpha
if (tau > 1e-9f)
    LPF_Init(&lpf, Ts / (Ts + tau));  // α = Ts/(Ts+τ)
else
    LPF_Init(&lpf, 1.0f);              // α = 1，直通不滤波
```

如果已知截止频率求 α：

```c
float fc = 100.0f;   // 想要 100Hz 截止频率
float alpha = (2.0f * 3.14159f * fc * Ts) / (1.0f + 2.0f * 3.14159f * fc * Ts);
LPF_Init(&lpf, alpha);
```

---

## 7. 核心记忆

| | τ 增大 | τ 减小 |
|---|---|---|
| α | ↓ 变小 | ↑ 变大 |
| $f_c$ | ↓ 变小 | ↑ 变大 |
| 滤波强度 | ↑ 强 | ↓ 弱 |
| 响应速度 | ↓ 慢 | ↑ 快 |
| 相位滞后 | ↑ 大 | ↓ 小 |

> **一句话**：τ 越大 = 滤波越狠 = α 越小 = 截止频率越低。一般从 $\tau = 10T_s$ 起步即可。
